# HingaSmart Rwanda

HingaSmart Rwanda is a simple smart agriculture web project built using **HTML** and **Tailwind CSS**.  
It includes a login page and a farmer dashboard.

---

##  How It Works

1. Open `Login form.html`
2. Click **Go to Dashboard**
3. It opens `Agri system.html`
4. Click **Logout** to return to login page

---

##  Pages Overview

### 1. Login form.html
- Username input
- Password input
- Keep me signed in checkbox
- Forgot password link (placeholder)
- Go to Dashboard button
- Create account link (placeholder)

---

### 2. Agri system.html
Includes:

- Navigation bar (My Farm, Market, Advice, Logout)
- Farmer Portal (rain, soil, season, harvest info)
- Food Market (products and prices)
- View More Products toggle
- Expert Advice alert section
- Footer

---

##  Technologies Used

- HTML5
- Tailwind CSS (CDN)
- Responsive design
- Frontend only (no backend)

---

##  Important Notes

- No real authentication system
- No database connection
- All data is static
- Internet connection required (for Tailwind CDN)
